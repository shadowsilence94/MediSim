# MediSim 🚀

A Group Project for NLP Class using Antigravity!

## About

MediSim is an NLP (Natural Language Processing) group project that embraces the spirit of Python's famous `antigravity` Easter egg. This project demonstrates the fun side of Python programming while building a foundation for serious NLP work.

## Features

- 🚀 Antigravity-powered launch system
- 📚 Built for NLP class group project
- 🐍 Pure Python implementation
- 🌟 Extensible architecture for NLP tasks

## Installation

```bash
# Clone the repository
git clone https://github.com/shadowsilence94/MediSim.git
cd MediSim

# Install the package
pip install -e .
```

## Usage

### Command Line

```bash
# Run MediSim
medisim
```

### Python API

```python
import medisim

# Launch MediSim with antigravity!
medisim.launch()
```

### Run the Demo

```bash
python examples/demo.py
```

## About Antigravity

The `antigravity` module is a famous Python Easter egg that references the XKCD comic about how Python makes programming feel effortless - like defying gravity! When you import antigravity, it opens a web browser to display the comic.

Learn more: [XKCD #353 - Python](https://xkcd.com/353/)
